# Fake-news-detection


#The project has 3 tabs also useful for the detection.
those are as follows:
1)Classify News : Predicts whether news is Fake or Real with decided Threshold
2)Batch Samples:Runs the model on multiple sample statements to see overall behavior for the current threshold and give visualization pie chart.
3)Model Insights : Model Performance Comparison (TF-IDF + LR vs LSTM) with visualization plots.
